Ticket to ride (University final project for the second semester of my first year)

Aventuriers du rail autour du monde (projet final pour le second semestre de BUT1)

Le projet s'est déroulé en 3 parties:
  • Une première partie où j'ai recréé la logique du jeu dans son entièreté;
  • Une deuxième où nous devions intégrer des algorithmes de graphes pour calculer les points et donner des indications aux joueurs;
  • Enfin, une troisième partie où il fallait créer de toute pièce l'interface graphique du jeu (IHM), avec le framework JavaFX.
This project was split into three parts:
  • The first one where we had to code the entire game logic
  • The second one where we had to research and integrate graph algorithms into our code to help with counting each player's points
  • Finally, a third one where we had to design and create the entire GUI of the game, using the JavaFX framework.
